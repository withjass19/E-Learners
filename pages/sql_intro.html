MySQL : An Introduction


SQL stands for “Structured Query Language” and can be pronounced as “SQL” or “SEQUEL”. It is a special-purpose language use to define, access, and manipulate data in RDBMS. SQL provides a set of statements for storing and retrieving data to and from a relational database. The original version called SEQUEL( Structured English Query Language ) was designed by IBM in mid-1970s. SQL was first introduced as a commercial database system in 1979 by Oracle Corporation.


Advantages of SQL :
  • SQL can retrieve data from a database.
  • SQL can execute queries against a database.
  • SQL can insert records in a database.
  • SQL can update records in a database.
  • SQL can delete records from a database.
  • SQL can create new databases.
  • SQL can create new tables in a database.
  • SQL can create store procedures in a database.
  • SQL can create views in a database.
  • SQL can set permissions on tables, procedures, and views.
SQL * Plus

SQL *Plus is a software product from Oracle Corporation that allows users to interactively use the SQL commands, produce formatted reports and support written command procedures to access Oracle databases.



SQL Commands

SL commands are instructions used to communicate with the database to perform specific task that work with data. SQL commands can be used not only for searching the database but also to perform various other functions like, for example you can create tables, add data to tables, or modify data, drop the table, set permissions for users.


SQL commands are grouped into five major categories depending on their functionality :

  • Data Query Language ( DQL ) : SQL command used for retrieving or selecting data from the database. DQL command used in SQL is SELECT.
  • Data Definition Language ( DDL ) : There SQL commands are used for creating, modifying, and dropping the structure of database objects. DDL commands used in SQL are CREATE, ALTER, DROP, RENAME, and TRUCATE.
  • Data Manipulation Language ( DML ) : These SQL commands are used for inserting, modifying, and deleting data. DML commands used in SQL are INSERT, UPDATE, and DELETE.
  • Transaction Control Language ( TCL ) : There SQK commands are used for managing changes affecting the data. TCL commands used in SQL are COMMIT, ROLLBACK, and SAVEPOINT.
  • Data Control Language ( DCL ) : These SQL commands are used for providing security to the database objects. DCL commands used in SQL are GRANT and REVOKE.
